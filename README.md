
# 🚀 BOTDETECT – Twitter Bot Detection System

## 📌 Project Overview

**BOTDETECT** is a hybrid deep learning framework designed to detect Twitter bot accounts using both:

* 📝 Textual Features (Tweets / Bio content)
* 📊 Behavioral Metadata (Followers, Following, Tweet Count, Account Age, etc.)

The system integrates multiple deep learning models and Explainable AI (XAI) to ensure transparency and reliability in predictions.

---

## 🎯 Problem Statement

With the rise of fake accounts and automated bots on social media, detecting genuine users has become crucial.

BOTDETECT aims to:

* Identify bot accounts
* Analyze behavioral patterns
* Use textual embeddings
* Provide explainable predictions

---

## 🧠 Models Implemented

The following deep learning models were implemented and compared:

* 🔹 RNN (Recurrent Neural Network)
* 🔹 CNN (Convolutional Neural Network)
* 🔹 DNN (Deep Neural Network)

Each model was evaluated using training and testing accuracy.
